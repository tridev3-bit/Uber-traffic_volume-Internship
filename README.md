# Traffic Volume Prediction using Machine Learning

## Project Overview
This project predicts hourly traffic volumes at different road junctions using historical traffic data.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Machine Learning Model
- Random Forest Regressor

## Project Workflow
- Data preprocessing
- Feature engineering
- Exploratory Data Analysis (EDA)
- Model training
- Model evaluation (MAE, RMSE, R² Score)
- Model refinement

## Results
Successfully built and improved a machine learning model to forecast hourly traffic volumes using historical data.
